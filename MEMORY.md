## מה עבד

### RAG מעשיר את קולות התיאורטיקנים
חיבור כל תיאורטיקן לבסיס ידע של טקסטים מקוריים שיפר משמעותית את עומק התגובות. הסוכן "מכיר" את הגישה ולא רק מחקה אותה.
תאריך: אפריל 2026

### Output validation loop
לולאה שאוכפת "שאלה אחת בלבד" ו"פתיחה מגוונת" שיפרה את איכות השיחות. מונעת את הדפוס הנפוץ של אנליטיקאי ששואל 3 שאלות בתגובה אחת.
תאריך: אפריל 2026

### Vercel-native crons במקום CCR external
מעבר מ-CCR agents שקוראים ל-Vercel API (נחסמו ב-403) ל-crons נטיביים ב-vercel.json שעובדים מבפנים.
כלל: CCR agents עובדים עם repo ישירות — לא קוראים ל-Vercel deployment.
תאריך: אפריל 2026

### פרסונות מדומות לפידבק UX
מיכל (33, מעצבת גרפית, 2.5 שנות טיפול) נותנת פידבק UX ספציפי הרבה יותר מ-QA כללי. הדמות עם ביוגרפיה, מטרה, ורמת טכנולוגיה — לא סוכן אנונימי.
תאריך: אפריל 2026

### כפתור "מה לקחתי מהשיחה" — נסתר עד 2+ תורות
כפתור שמופיע רק אחרי שהשיחה התחילה. נכון מבחינת UX — לא להציג כלי שלא רלוונטי עדיין.
תאריך: אפריל 2026

## מה לא עבד

### External CCR → Vercel API = 403 Forbidden
**מה ניסינו:** Python scripts מ-CCR agents שקוראים ל-psychoanalytic-space.vercel.app
**מה קרה:** Vercel חוסמת בקשות חיצוניות ב-403
**הפתרון:** Vercel-native crons (vercel.json) שקוראים לעצמם מבפנים
תאריך: אפריל 2026

### Gender bug בסימולציות
**מה ניסינו:** שימוש בפרסונה נשית ללא הנחיות מפורשות על מגדר
**מה קרה:** Claude כתב בלשון זכר (יודע, חושב, לובש) במקום נקבה
**הפתרון:** CRITICAL GENDER block עם CORRECT/WRONG examples ו-SELF-CHECK בכל prompt סימולציה
תאריך: אפריל 2026

### Stream idle timeout ב-CCR
**מה ניסינו:** ייצור 4 סימולציות מלאות בתגובה אחת
**מה קרה:** timeout — הסוכן נקטע באמצע
**הפתרון:** כל שלב נשמר לקובץ זמני (/tmp) לפני המעבר לשלב הבא. כתוב → שמור → כתוב → שמור
תאריך: אפריל 2026

### supabaseClient undefined
**מה ניסינו:** קריאה ל-supabaseClient.auth בלחיצה על כפתור
**מה קרה:** TypeError — CDN של Supabase לא תמיד נטען לפני הלחיצה
**הפתרון:** guard `if (!supabaseClient)` בתחילת signIn/signUp/resetPassword
תאריך: אפריל 2026

### updateReflectionBtn לא נקרא
**מה ניסינו:** כפתור "מה לקחתי" הוגדר אבל לא הופיע
**מה קרה:** הפונקציה לא נקראה אחרי עדכון השיחה
**הפתרון:** updateReflectionBtn() נקרא ב-4 מקומות: אחרי כל תגובת AI, אחרי restoreConversation, ובinit
תאריך: אפריל 2026
